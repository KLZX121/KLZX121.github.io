About: I made this for myself to improve my Morse code skills, hence why it is really roughly put together. It basically gives you a random word (or letter/number depending on the mode you have chosen) and you translate it in Morse, or vice versa. A Morse chart can be found here.

Settings

Languages:

  • English: English Output - Translate to Morse
  • Morse: Morse Output - Translate to English
Types:
  • Words: Outputs a word from a list of the 1000 most common English words (Credits)
  • Letters: A single letter from the English alphabet
  • Numbers: A number from 0 - 9 if iterating or a number from 0 - 999999 if random
Modes:
  • Random: Chooses from the selected list randomly
  • Iterate: Runs through the selected list one by one

Hotkeys:

Space = short press ( dot · ) / long press ( dash − )

, (Comma) = space

Enter = submit

LControl = skip word

Backspace = delete last character
