About: I made this for myself to improve my Morse code skills, hence why it is really roughly put together. It basically gives you a random word (or letter depending on the mode you have chosen) and you use the hotkeys listed to type it out in Morse. A Morse chart can be found here.

Languages:

  • English: Outputs the phrase in English
  • Morse: Outputs the phrase in Morse
Types:
  • Words: Outputs a word from a list of the 1000 most common English words (Credits)
  • Letters: A single letter from the English alphabet
  • Numbers: A number from 0 - 9 if iterating or a number from 0 - 999999 if random
Modes:
  • Random: Chooses from the selected list randomly
  • Iterate: Runs through the selected list one by one
